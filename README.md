# My Chat App 🚀

Một ứng dụng chat thời gian thực mạnh mẽ được xây dựng với Django, React và WebSockets.

## ✨ Tính năng chính
- Chat Real-time với WebSockets (Django Channels).
- Xác thực JWT an toàn.
- Quản lý trạng thái Online/Offline.
- Lưu trữ lịch sử tin nhắn với PostgreSQL.

## 🛠 Công nghệ sử dụng
- **Backend:** Django, Django Rest Framework, Channels.
- **Frontend:** React, Tailwind CSS.
- **Infrastructure:** Redis, PostgreSQL, Docker.

## 🚀 Hướng dẫn cài đặt nhanh (Docker)
1. Clone dự án: `git clone https://github.com/ngocson2g/my-chat-app.git`
2. Chạy lệnh: `docker-compose up --build`
3. Truy cập: `localhost:3000` (Frontend) và `localhost:8000/api/docs` (Swagger).
